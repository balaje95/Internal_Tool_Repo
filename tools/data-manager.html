Zuper Workflow Builder export, or a raw JobNimbus automations export (auto-converted)

Imports straight into the currently connected account's Workflow Builder via workflows/import. Each workflow in the file is created as a brand-new workflow (fresh UID) — it never overwrites an existing one.
JobNimbus files are detected automatically and converted to Zuper's format on the fly — review the warnings shown in the preview step before importing, since some fields (user/status/category UIDs) can't be auto-mapped and need manual follow-up.

These converted workflows contain [MAP → …] placeholders that need a real value from this account before they'll run. Pick a value (or type one) for each. Leave a row blank to import the placeholder as-is and fix it later in the Workflow Builder.
