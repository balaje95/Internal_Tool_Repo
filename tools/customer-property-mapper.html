🏠 Zuper Customer ↔ Property Mapper

Fetch properties & customers, then bulk-map customers to each property

1 Connect to a Zuper account

Enter the account's API key. The data-center region is detected automatically (same as the other tools).

2 Fetch properties & customers

Pulls every property and customer in the account, and reads which customers are already linked to each property.

3 Map customers to properties

Each row is a property. Add one or more customers to a property — by dropdown, bulk tools, or a comma-separated import. On apply, the staged customers are merged into the property's customer list (existing customers are preserved).
